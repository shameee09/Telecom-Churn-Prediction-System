# 📊 Telecom Churn Prediction System

An intelligent web-based application that predicts customer churn in the telecom industry using Machine Learning and provides AI-driven retention strategies.

## 🚀 Features

- 🔐 User Authentication (Admin & Customer)
- 📥 Customer Data Submission Form
- 🤖 Churn Prediction using ML model (Scikit-learn)
- 📈 Admin Dashboard with analytics & charts
- 📊 Risk Classification (Low / Medium / High)
- 💡 AI Retention Suggestions using Groq LLM
- 📋 Customer Management Panel

## 🧠 Tech Stack

- **Frontend:** HTML, CSS, JavaScript
- **Backend:** Flask (Python)
- **ML Model:** Scikit-learn
- **Database:** SQLite
- **AI Integration:** Groq (LLM API)
- **Visualization:** Chart.js

## 📊 Workflow

1. Customer submits telecom details  
2. ML model predicts churn probability  
3. System classifies risk level  
4. Admin views analytics dashboard  
5. AI generates retention strategies for high-risk customers  

## 🎯 Key Highlights

- Real-time prediction system  
- AI-powered decision support  
- Clean UI with analytics dashboard  
- Role-based access (Admin / Customer)
